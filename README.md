Flask Template
-----------------------------------
This is how I initiate you into Python web development with Flask and please start with Python 3 :)

# Install
```bash
$ virtualenv venv
$ . venv/bin/activate
(venv)$ pip install -r requirements.txt
```

# Add/Remove deps
```bash
(venv)$ pip freeze > requirements.txt
```

# Run locally
```bash
(venv)$ python app.py
```
